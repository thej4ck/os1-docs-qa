Configurazione esterna eCommerce B2C

Per configurare un'azienda per gestire il riallineamento dei dati con un sito web di eCommerce è necessario creare nella cartella Custom una cartella eCommerce e copiare dalla cartella Modelli\eCommerce il file 'B2C_ReplicaWebConfig.ini'.

Il file copiato deve essere rinominato sostituendo il prefisso 'B2C' con il codice ditta riportando 'B2C' alla fine del nome. Ad esempio: DWEB_ReplicaWebConfigB2C.ini.

Nella stessa cartella Custom\eCommerce deve essere presente anche il file di licenza generato tramite il programma XNSerie il cui nome deve essere dato dal codice della ditta seguito dal suffisso '_eCommerceB2C.licweb'. Ad esempio: DWEB_eCommerceB2C.licweb.

 

Struttura del file di configurazione

Il file ha parametri specifici e contiene anche delle righe di commento, precedute dal carattere ';' che spiegano dettagliatamente il significato ed i formati delle righe presenti.

 

[Modulo]

Modulo=EB2C

 

[Immagini]

TipiImg=.jpg

;   TabellaWeb=TabellaOS1;TipoGestione;Chiave;CampiEsclusi;Filtro;

;      0 = Gestione standard OS1->Web

;    100 = Gestione standard Web>Os1

; 4.Chiave = Elenco campi chiave primaria separati da ","
