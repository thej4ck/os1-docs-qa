Configurazione esterna lettori barcode

Per configurare un lettore barcode è necessario creare nella cartella Custom una cartella Barcode e copiare dalla cartella Modelli\Barcode il file relativo al lettore barcode utilizzato.

Il file copiato deve essere rinominato con nome macchina seguito dal suffisso '_BCODE' e deve avere estensione cfg. Ad esempio: MIOPC_BCODE.cfg

Attualmente sono gestiti i seguenti lettori barcode:

      • DataLogic Formula 734

      • DataLogic MemorX3

      • Meteor ECO 486

      • Meteor Sirio2000

      • Meteor Lion Zigbee

      • Denso BHT-8000

 

Struttura dei file di configurazione

Ogni file ha parametri specifici e contiene anche delle righe di commento, precedute dal carattere ';' che spiegano come effettuare gli eventuali settaggi.

 

DataLogic Formula 734

[Config]

;DataLogic - Formula 734 Laser

Port=COM1 [Porta di comunicazione da utilizzare]

Parametri=baud=9600 parity=M data=7 stop=1 [Settaggio della porta]

Modulo=OS1BC_DLG [nome del modulo .bpl]

 

DataLogic MemorX3

Per utilizzare questo tipo di lettore è necessario installare il software a corredo RXAS (RX Active Sync v.2.2.7).

 

[Config]

;Datalogic - MemorX3

;Il parametro Port non deve essere lasciato vuoto

Port=COM1 [parametro non significativo, ma obbligatorio]

;Impostare il separatore di campo sul lettore portatile

Separatore=| [carattere separatore di lista da impostare anche sul lettore]

LenCodice= [è obbligatorio, in assenza di separatore, la dimensione del barcode]

Parametri=C:\DataLogic\Dati.txt [nome completo di percorso del file proveniente dal lettore tramite RXAS]

SeparatoreEANGS1=]

Modulo=OS1BC_MET [nome del modulo .bpl]

 

Meteor ECO 486

Per utilizzare questo tipo di lettore è necessario installare il software a corredo RSFTerminal.

 

[Config]

;Meteor - ECO 486

;Il parametro Port non deve essere lasciato vuoto

Port=COM1 [parametro non significativo, ma obbligatorio]

;Impostare il separatore di campo sul lettore portatile

Separatore=; [carattere separatore di lista da impostare anche sul lettore]

Parametri=C:\RSFTerminal\Eacq.dat [nome completo di percorso del file proveniente dal lettore tramite RSFTerminal]

Modulo=OS1BC_MET [nome del modulo .bpl]

 

Meteor Sirio2000

Per utilizzare questo tipo di lettore è necessario installare il software a corredo EacqCE Daemon v.2.1.0.

 

[Config]

;Meteor - Sirio2000

;Il parametro Port non deve essere lasciato vuoto

Port=COM1 [parametro non significativo, ma obbligatorio]

;Impostare il separatore di campo sul lettore portatile

Separatore=| [carattere separatore di lista da impostare anche sul lettore]

LenCodice= [è obbligatorio, in assenza di separatore, la dimensione del barcode]

Parametri=C:\EacqCE_Daemon\EacqCE_Data.txt [nome completo di percorso del file proveniente dal lettore tramite EacqCE Daemon]

SeparatoreEANGS1=]

Modulo=OS1BC_MET [nome del modulo .bpl]

 

Meteor Lion Zigbee

[Config]

;METEOR - Lion Zigbee

;Il parametro Port non deve essere lasciato vuoto

Port=COM1 [parametro non significativo, ma obbligatorio]

;Impostare il separatore di campo sul lettore portatile

Separatore= [parametro non significativo]

LenCodice= [parametro non significativo]

Parametri= [parametro non significativo]

SeparatoreEANGS1=]

Modulo=OS1BC_MET [parametro non significativo, ma obbligatorio]

 

Denso BHT-8000

Per utilizzare questo tipo di lettore è necessario installare il software a corredo BHT Manager.

 

[Config]

;DENSO - BHT8000

;Il parametro Port non deve essere lasciato vuoto

Port=COM1 [parametro non significativo, ma obbligatorio]

;Impostare il separatore di campo sul lettore portatile solo se presente altrimenti impostare LenCodice

Separatore=  [il programma Wizard del lettore di solito non prevede separatore di lista]

LenCodice=13 [è obbligatorio, in assenza di separatore, la dimensione del barcode]

Parametri=C:\DENSO\Dati.dat [nome completo di percorso del file proveniente dal lettore tramite BHT Manager]

SeparatoreEANGS1=]

Modulo=OS1BC_MET [nome del modulo .bpl]
