Creazione database di transito

La procedura consente di creare il database di transito necessario in fase di riallineamento dati con il Web.

 

 

TIPO ECOMMERCE: tipo di installazione eCommerce utilizzato. La scelta è condizionata ai tipi di eCommerce presenti nella licenza OS1.

NOME DATABASE: nome del database di transito da creare; viene proposto il nome del database della ditta corrente seguito dalla dicitura '_Transito'.

PASSWORD 'sa': password dell'utente sa (system administrator) del server SQL al quale siamo collegati; viene richiesta solo se si è collegati con un utente diverso.
