Configurazione esterna eCommerce B2B

Per configurare un'azienda per gestire il riallineamento dei dati con un sito web di eCommerce è necessario creare nella cartella Custom una cartella eCommerce e copiare dalla cartella Modelli\eCommerce il file 'B2B_ReplicaWebConfig.ini'.

Il file copiato deve essere rinominato sostituendo il prefisso 'B2B' con il codice ditta. Ad esempio: DWEB_ReplicaWebConfig.ini.

Nella stessa cartella Custom\eCommerce deve essere presente anche il file di licenza generato tramite il programma XNSerie il cui nome deve essere dato dal codice della ditta seguito dal suffisso '_eCommerce.licweb'. Ad esempio: DWEB_eCommerce.licweb.

 

Struttura del file di configurazione

Il file ha parametri specifici e contiene anche delle righe di commento, precedute dal carattere ';' che spiegano dettagliatamente il significato ed i formati delle righe presenti.

 

[Modulo]

Modulo=EB2B [valore fisso]

 

[Immagini]

TipiImg=.jpg [elenco estensioni immagini utilizzate separate dal carattere ';' (attualmente sono gestiti solo i formati jpg)]
