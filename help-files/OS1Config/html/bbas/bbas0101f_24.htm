Configurazione offerte clienti

La compilazione di questa tabella è richiesta agli utenti che utilizzano il modulo Gestione Offerte Clienti.

DATA RICHIESTA DI TESTA: Abilita l’inserimento, in gestione offerte, della data di richiesta consegna di testa (casella con segno di spunta).

DATA PRODUZIONE DI TESTA: Abilita l’inserimento, in gestione offerte, della data di produzione di testa (casella con segno di spunta).

DATA CONSEGNA DI TESTA: Abilita l’inserimento, in gestione offerte, della data di prevista consegna di testa (casella con segno di spunta).

DATA RICHIESTA DI RIGO: Abilita l’inserimento, in gestione offerte, della data di richiesta consegna di rigo (casella con segno di spunta).

DATA PRODUZIONE DI RIGO: Abilita l’inserimento, in gestione offerte, della data di produzione di rigo (casella con segno di spunta).

DATA CONSEGNA DI RIGO: Abilita l’inserimento, in gestione offerte, della data di prevista consegna di rigo (casella con segno di spunta).

GESTIONE MISURE PRODOTTI: Definisce se in gestione offerte è attivo l'inserimento delle misure del prodotto (lunghezza, larghezza, altezza) al fine di calcolare la quantità.

IMPORTA DESCRIZIONE ESTESA: Se attivo in fase di gestione offerta per i prodotti la cui descrizione estesa non è fissa viene riportata la descrizione estesa del prodotto nel primo campo di annotazioni disponibile sul rigo in modo tale da poterlo modificare.

TIPO RIGO SUBTOTALI: Indica il tipo rigo riservato per le righe documento di tipo subtotale. Tale tipo rigo viene utilizzato per inserire sull’offerta un subtotale che può essere gestito come totale alternativo o come totale intermedio, in base a quanto impostato sulla causale utilizzata per emettere il documento.

TIPO RIGO NORMALE PROD. PROVV.: Indica il tipo rigo riservato per le righe normali che trattano prodotti provvisori.

TIPO RIGO OMAGGIO PROD. PROVV.: Indica il tipo rigo riservato per le righe omaggio che trattano prodotti provvisori.

TIPO RIGO NORMALE PROD. DEFIN.: Indica il tipo rigo riservato per le righe normali che trattano prodotti provvisori; utilizzato in fase di definizione prodotti e/o clienti provvisori per aggiornare le righe delle offerte con il tipo rigo corretto.

TIPO RIGO OMAGGIO PROD. DEFIN.: Indica il tipo rigo riservato per i righi omaggio che trattano prodotti provvisori; utilizzato in fase di definizione prodotti e/o clienti provvisori per aggiornare le righe delle offerte con il tipo rigo corretto.

NUMERO GIORNI PER VALIDITÀ: Indica il numero dei giorni da utilizzare per il calcolo della data di validità da proporre in emissione offerta a partire dalla data in cui viene inserito il documento.

CONTROLLO VALIDITÀ IN EVASIONE: Indica se si desidera effettuare un controllo di validità dell’offerta in fase di evasione di quest’ultima; valori ammessi:

    • Nessun controllo: non effettua nessun controllo.

    • Controllo con data del documento: considera solo le offerte con data validità maggiore o uguale alla data del documento di evasione.

    • Controllo con data di sistema: considera solo le offerte con data validità maggiore o uguale alla data di sistema.

RIF. SU DOCUMENTI: Definisce se si desidera inserire un rigo descrittivo, in fase di evasione offerta, che indica il documento evaso, valori ammessi:

    • Nessun riferimento: non genera alcun rigo descrittivo.

    • Riferimento interno: genera rigo descrittivo contenente l’indicazione della causale, del numero interno dell’offerta e della data dell’offerta.

    • Riferimento esterno: genera rigo descrittivo contenente il riferimento inserito nel campo “Vs. richiesta” sulla testa dell’offerta.

    • Entrambi i riferimenti: genera rigo descrittivo con entrambi i riferimenti.

DESCRIZIONE PER DOCUMENTO: Breve descrizione, utilizzata in fase di evasione offerta nel caso in cui si desideri inserire il riferimento; precede il riferimento all’offerta stessa.

TIPO CONTROLLO DISPONIBILITÀ: Indica il tipo di controllo da effettuare in fase di emissione offerta clienti sulle disponibilità di magazzino. Tale controllo, se attivo, può essere vincolante o meno e può essere attivato solo sul magazzino indicato sul rigo del movimento oppure su tutti i magazzini di proprietà dell’azienda.

Richiesta criteri di selezione in evasione: Definisce se richiedere, in fase di evasione, dei criteri di selezioni supplementari per la selezione dei documenti da evadere (casella con segno di spunta).

Consenti evasione delle sole offerte accettate: Se spuntato consente di evadere solo le offerte che sono state accettate.

Controllo prezzo in offerta: Se attivo in fase di inserimento offerta viene segnalato se il rigo ha prezzo uguale a zero.
