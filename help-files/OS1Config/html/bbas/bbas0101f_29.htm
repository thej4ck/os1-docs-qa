Configurazione gestione varianti

La compilazione di questa tabella è richiesta agli utenti che utilizzano il modulo Gestione varianti.

GESTIONE VARIANTI: Abilita la gestione delle varianti a livello generale dell’applicazione.

CODICE VARIANTE BASE: Indica il codice variante base che viene utilizzata in fase di inserimento dettaglio quantità per quei prodotti per cui non viene inserita la variante.

Gestione prezzi per variante: Indica se è attiva la gestione dei prezzi per variante (casella con segno di spunta); se la gestione è attiva, per i prodotti che gestiscono la variante sarà possibile definire se il prezzo deve essere gestito per prodotto o per prodotto/variante. Per i prodotti gestiti a variante che utilizzano i prezzi per variante, sul dettaglio dei documenti/movimenti sarà possibile inserire una sola variante. Se si attiva la gestione dei prezzi per variante, è necessario compilare il campo 'Data inizio gestione prezzi per variante' (campo successivo), altrimenti la gestione sarà disattivata. Se esistono prodotti che gestiscono la variante ed i prezzi per variante questo campo non sarà modificabile.

Data inizio gestione prezzi per variante: Indica la data dalla quale sarà attiva la gestione dei prezzi per variante; se un prodotto ha documenti antecedenti a questa data ancora da evadere non sarà possibile attivare la gestione prezzi per variante. Per i prodotti che gestiscono i prezzi per variante la modifica di documenti antecedenti a questa data consentirà l'utilizzo del dettaglio con più varianti e la variante non influirà sulla ricerca del prezzo. Sul dettaglio dei documenti/movimenti emessi in questa data o successivamente, per i prodotti gestiti a variante che utilizzano i prezzi per variante, sarà possibile inserire una sola variante. Se è presente la spunta sul campo 'Gestione prezzi per variante' (campo precedente), è necessario compilare questo campo altrimenti la gestione sarà disattivata. Se esistono prodotti che gestiscono la variante ed i prezzi per variante questo campo non sarà modificabile.
