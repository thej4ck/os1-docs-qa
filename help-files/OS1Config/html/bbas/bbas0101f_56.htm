Configurazione Pagamenti fornitori

CAUSALE CVS: Causale di default proposta in fase di elaborazione pagamenti a fornitori esteri; la causale indicata sul pagamento sarà utilizzata per la generazione del flusso relativo ai bonifici esteri.

TIPO CVS: Valore di default proposto in fase di elaborazione pagamenti a fornitori esteri; il valore indicato sul pagamento sarà utilizzato per la generazione del flusso relativo ai bonifici esteri.

Cartella flussi Avvisatura Effetti: Definisce la cartella predefinita che viene proposta in fase di lettura dei files di Avvisatura Effetti.

Versione file XML CBI pagamenti Italia: Il campo viene utilizzato per la compilazione dell'intestazione e dei contenuti del file XML in fase di generazione da scadenze di tipo Bonifico verso fornitori italiani relativamente ai conti correnti che utilizzano il formato XML CBI; valori possibili Versione 00.04.00, Versione 00.04.01.

Vers. file XML ISO20022 pagamenti Italia: Il campo viene utilizzato per la compilazione dell'intestazione e dei contenuti del file XML in fase di generazione da scadenze di tipo Bonifico verso fornitori italiani relativamente ai conti correnti che utilizzano il formato XML ISO 20022; valori possibili: Versione pain.001.001.03, Versione pain.001.001.09.
