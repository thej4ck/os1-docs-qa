Configurazione

La tabella consente la memorizzazione delle credenziali per connettersi al servizio CreditShield; tramite il pulsante "Verifica credenziale" è possibile effettuare un test di validità sulle credenziali inserite; la casella "Aggiornamento semafori automatico", se spuntata consente di indicare che per l'azienda deve essere effettuato l'aggiornamento automatico dei semafori.
