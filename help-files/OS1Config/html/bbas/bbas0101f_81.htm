Configurazione Arxivar Next

CLIENTID: Il ClientID è reperibile da ArxivarNext tramite l'interfaccia ARXivarNextAuthentication (esempio http//MioServer/ARXivarNextAuthentication) nella sezione "Software".

CLIENTSECRET: Il ClientSecret è reperibile da ArxivarNext tramite l'interfaccia ARXivarNextAuthentication (esempio http//MioServer/ARXivarNextAuthentication) nella sezione "Software". Deve essere indicato il valore corrispondente al ClientID selezionato.

URL base per i servizi API: indicare il nome del server in cui è installato Arxivar; esempio: http://MioServer/ARXivarNextWebApi/Api/.
