Configurazione default cartelle

La compilazione di questa tabella interessa gli utenti che utilizzano le procedure di scambio dati elettronici con le banche.

CARTELLA FLOPPY RI.BA: Definisce la cartella predefinita che viene proposta in fase di generazione del file RI.BA. elettroniche.

CARTELLA FLOPPY RID: Definisce la cartella predefinita che viene proposta in fase di generazione del file RID elettronici.

CARTELLA FLOPPY MAV: Definisce la cartella predefinita che viene proposta in fase di generazione del file MAV elettronici.

Cartella flussi bonifici fornit./ritiro effetti: Definisce la cartella predefinita che viene proposta in fase di generazione del file bonifici a fornitori.

CARTELLA STAMPA OFFERTE WORD: Definisce la cartella predefinita che viene utilizzata per la memorizzazione delle offerte stampate direttamente in Word.

CARTELLA STAMPA ORDINI FORNITORI WORD: Definisce la cartella predefinita che viene utilizzata per la memorizzazione degli ordini fornitori stampati direttamente in Word.

Cartella stampa richieste off. Word: Definisce la cartella predefinita che viene utilizzata per la memorizzazione delle offerte stampate direttamente in Word.

Cartella esportazione dati IVA di gruppo: Definisce la cartella predefinita che viene utilizzata per l'esportazione dei dati relativi alle liquidazioni IVA ed ai progressivi IVA nel caso si gestisca il gruppo IVA.
