Configurazione File Store

NOME SERVER: è il nome del PC su cui viene eseguito il servizio a cui si collega OS1 per l'archiviazione degli allegati.

PORT: è il numero della porta TCP-IP del PC su cui viene eseguito il servizio a cui si collega OS1 per l'archiviazione degli allegati.

COMPRESSIONE DOCUMENTI: se spuntato gli allegati vengono compressi prima di essere salvati nel database.

SCANSIONE IN FORMATO PDF: se spuntato i documenti acquisiti tramite scanner generano sempre un file PDF, altrimenti viene generato un file TIF multipagina.
