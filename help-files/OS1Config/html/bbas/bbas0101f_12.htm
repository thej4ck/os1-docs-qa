Configurazione distinta base

La compilazione di questa tabella è richiesta agli utenti che utilizzano il modulo Distinta base.

LAVORANTE SU SCHEDA LAVORAZ.: Indica se in fase di gestione delle schede di lavorazione può essere indicato anche il codice del lavorante; utilizzato dalle procedure del modulo distinta base

GESTIONE DATA VALIDITÀ: Se attivo le distinte base inserite vengono gestite per data di decorrenza per cui una volta resa definitiva una distinta, qualsiasi modifica che venga apportata, determina la storicizzazione della versione precedente e l'entrata in vigore della nuova distinta base.

LIVELLO MASSIMO DB: Indica il numero massimo di nidificazioni permesse nella struttura della distinta base.

GESTIONE DISTINTA NEUTRA: Indica se si gestisce la distinta base variabile, contenente cioè componenti neutri che di volta in volta devono essere sostituiti con componenti specifici; utilizzato in fase di gestione distinta base ed in gestione prima nota di magazzino.

GESTIONE SFRIDO IN DISTINTA: Se spuntato attiva la gestione dell’indicatore di sfrido sui componenti della distinta base.

Utilizza prezzo forn. abituale in valorizzazione: In fase di valorizzazione tramite listino, se non viene trovato nessun prezzo valido, sarà effettuata una ulteriore ricerca del prezzo su contratti e prezzi speciali utilizzando il fornitore abituale (casella con segno di spunta).

Utilizza perc. valore su documenti: Se spuntato, in fase di esplosione dei componenti da distinta base nella gestione documenti, il prezzo dei singoli componenti verrà calcolato rapportando l'importo netto del composto in base alle percentuale presente sulla distinta base (campo 'Perc. valore').

Segnala componenti variabili: Definisce se effettuare la segnalazione per la presenza di componenti variabili quando viene esplosa la distinta base di un composto, valori possibili:

      • non effettuare segnalazione

      • segnalazione solo in prima nota di magazzino

      • segnalazione in prima nota di magazzino e su documenti
