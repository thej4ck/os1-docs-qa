Configurazione Statistiche

CALCOLO VALORE LISTINO: se spuntato viene calcolato il valore del movimento in base al prezzo di listino.

Trattamento fatture proforma: indica il trattamento delle fatture proforma; valori possibili:

      • Tratta comunque.

      • Tratta solo se da evadere.

      • Non trattare.

Considera quantità annullate ordini clienti: Se attiva questa opzione (casella con segno di spunta) sarà utilizzata la quantità annullata presente sul rigo dell'ordine cliente per decrementare la quantità ordinata.

Considera quantità a saldo ordini clienti: Se attiva questa opzione (casella con segno di spunta) sarà utilizzata la quantità a saldo presente sul rigo dell'ordine cliente per decrementare la quantità ordinata.

Considera quantità annullate ordini fornitori: Se attiva questa opzione (casella con segno di spunta) sarà utilizzata la quantità annullata presente sul rigo dell'ordine fornitore per decrementare la quantità ordinata.

Considera quantità a saldo ordini fornitori: Se attiva questa opzione (casella con segno di spunta) sarà utilizzata la quantità a saldo presente sul rigo dell'ordine fornitore per decrementare la quantità ordinata.
