Configurazione OS1Mobile - Raccolta ordini

MANTIENI NUMERAZIONE DOCUMENTI: Definisce se in fase di importazione documenti, il documento deve mantenere il numero originale inserito all'interno di OS1Mobile (casella con segno di spunta), oppure deve essere rinumerato (casella vuota). Nella caso in cui si voglia mantenere la numerazione originale è necessario utilizzare per ogni agente causali che utilizzano serie diverse.

N. GIORNI INCR. CALCOLO CONSEGNA: Indica il numero dei giorni da utilizzare, per il calcolo della data di consegna, a partire dalla data in cui viene inserito il documento.

Assegna note ordine a note Ddt: Se spuntato le note del rigo ordine vengono riportate anche nel campo 'Note ddt'.

Assegna note ordine a note fattura: Se spuntato le note del rigo ordine vengono riportate anche nel campo 'Note fattura'.

Magazzino centrale: E' il codice del magazzino principale da utilizzare per la generazione dei documenti.
