Configurazione Standard - BoxFatture - Acquisizione fatture

Prodotto generico: Deve essere un prodotto di tipo 'Prestazione/Servizio'; utilizzato in fase di acquisizione fatture nel caso in cui non si riesca a determinare il codice del prodotto, se non compilato sarà generato un tipo rigo 'Prodotto descrittivo'.

Utilizza ricerca prodotto: Attiva la ricerca prodotto standard per il codice prodotto presente nel documento XML (casella con segno di spunta).
