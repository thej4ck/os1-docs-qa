Configurazione esterna OS1Mobile

Per configurare un'azienda per gestire il riallineamento dei dati con dispositivi mobili tramite concentratore è necessario creare nella cartella Custom una cartella Mobile e copiare dalla cartella Modelli\Mobile il file 'DDDD_ReplicaConcentratoreConfig.ini'.

Il file copiato deve essere rinominato sostituendo il prefisso 'DDDD' con il codice ditta. Ad esempio: DMOB_ReplicaConcentratoreConfig.ini.

Nel caso ci siano tabelle o campi personalizzati da gestire è necessario riportare nella stessa cartella Custom\Mobile anche il file script di creazione del database del concentratore (DBConcentratore.sql) opportunamente modificato per la creazione delle tabelle/campi personalizzati; inoltre se presenti tabelle personalizzate è necessario modificare anche il file DDDD_ReplicaConcentratoreConfig.ini per definire il metodo di scambio delle tabelle personalizzate.

 

Struttura del file di configurazione

Il file ha parametri specifici e contiene anche delle righe di commento, precedute dal carattere ';' che spiegano dettagliatamente il significato ed i formati delle righe presenti.

 

[TabellaDispositivi]

; Struttura:

;

; 1.Tabella Concentratore= Nome tabella sul database Concentratore

; 2.Tabella OS1= Nome tabella sul database  di OS1

[TabellaConfigurazioneWS]

; Struttura:

;

; 1.Tabella Concentratore

; 2.Function OS1= Function che ritorna il valore dal database di OS1

; 3.Filtro ( Valori da ricercare composti da MODULO . IDMODULO . CHIAVE )
