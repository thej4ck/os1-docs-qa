Commesse

 

CODICE: codice alfanumerico di 15 caratteri per identificare la commessa in esame. Sul campo sono attive le funzioni di ricerca (F9) sulla tabella stessa.

DESCRIZIONE: campo alfanumerico di 50 caratteri per descrivere la commessa.

NOTE: campo note su più righe per inserire eventuali annotazioni in merito all’utilizzazione della commessa.

DATA INIZIO: indica la data di inizio validità della commessa.

DATA FINE: indica la data di fine validità della commessa.

ORDINE DI STAMPA: indica il numero d'ordine valido per l'ordinamento in fase di stampa.

OBSOLETO: flag attivabile dall’utente per inibire il possibile utilizzo dell’elemento di tabella in esame nelle successive transazioni.
