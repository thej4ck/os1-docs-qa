Voci di costo/ricavo

 

CODICE: codice alfanumerico di 15 caratteri per identificare la voce di costo/ricavo in esame. Sul campo sono attive le funzioni di ricerca (F9) sulla tabella stessa.

DESCRIZIONE: campo alfanumerico di 50 caratteri per descrivere la voce C/R.

NOTE: campo note su più righe per inserire eventuali annotazioni in merito all’utilizzazione della voce C/R.

TIPO VOCE DI COSTO/RICAVO: indica la tipologia di appartenenza della voce corrente. Sul campo sono attive le funzioni di ricerca (F9) ed accesso interattivo alla tabella collegata (CTRL+W).

TIPO CONTO: definisce se la voce interessa Costi o Ricavi.

ORDINE DI STAMPA: indica il numero d'ordine valido per l'ordinamento in fase di stampa.

DATA CREAZIONE: indica la data in cui il codice è stato caricato. Al momento del caricamento di un nuovo record in questo campo viene proposta la data di sistema.

DATA OBSOLESCENZA: indica la data in cui il codice è stato dichiarato obsoleto.

OBSOLETO: flag attivabile dall’utente per inibire il possibile utilizzo dell’elemento di tabella in esame nelle successive transazioni. Spuntando la casella sarà automaticamente assegnata alla data obsolescenza la data di sistema.
