Testi note per richieste di offerta

La tabella consente di creare, individuandole in base ad un codice alfanumerico di 8 caratteri, descrizioni ricorrenti anche su più righe da utilizzare in sede di emissione richieste di offerta nelle note di testa, di coda o interne.

 

CODICE TESTO: codice alfanumerico di 8 caratteri per identificare il testo fisso. Sul campo sono attive le funzioni di ricerca (F9) sulla tabella stessa.

OGGETTO: campo alfanumerico di 50 caratteri la descrizione sintetica del testo fisso. Questa descrizione viene visualizzata sulle finestre di ricerca, ma non viene riportata sul documento.

TESTO DOCUMENTO: campo note per descrivere il testo, che sarà riportato nel rispettivo campo note del documento.

OBSOLETO: flag attivabile dall’utente per inibire il possibile utilizzo dell’elemento di tabella in esame nelle successive transazioni.
