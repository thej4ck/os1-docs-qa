Tipi imballaggi

La tabella consente di definire le tipologie di imballaggio da inserire sulla scheda del prodotto, che sarà riportato anche sulle righe dei documenti.

 

Finestra di input in formato dettaglio

Finestra di input in formato tabella

 

CODICE: campo alfanumerico di tre caratteri per inserire il codice del tipo imballaggio in esame. Sul campo sono attive le funzioni di ricerca sulla tabella stessa.

DESCRIZIONE: campo alfanumerico di 50 caratteri per descrivere la tipologia di imballaggio in esame.
