Avviamento

La procedura Fatturazione, che fa parte del ciclo passivo aziendale, utilizza praticamente tutte le tabelle e gli archivi Gestionali. E’ richiesta comunque la seguente sequenza di attivazione:

  • Controllo e completamento tabelle gestionali

  • Configurazione  Base

  • Configurazione  Dati preferenziali

  • Configurazione Conti fissi

  • Configurazione  Stampe

  • Configurazione  Standard

  • Configurazione  Acquisti

  • Tabelle Acquisti – Causali fatture

È inoltre del tutto evidente la necessità di disporre, al momento dell’inserimento di un documento, dei fornitori, degli articoli, delle modalità di pagamento, ecc., per le quali si rimanda ai corrispondenti capitoli e paragrafi del presente manuale.

Ovviamente dovrebbero essere create solo le tabelle utilizzate: se un utente non opera con l’estero, è del tutto superfluo che provveda a caricare le descrizioni degli articoli in lingua estera.

Ricorrendo l’esigenza, occorre creare le seguenti tabelle:

    • Listini fornitori

Se l’utente utilizza la procedura di Gestione magazzino ed intende effettuare i carichi di magazzino automaticamente all’atto dell’acquisizione di una fattura, prima della creazione delle causali di fatturazione è necessario che esistano le Causali di magazzino.

Durante l’esposizione, si fa spesso riferimento a programmi della Gestione Magazzino, per i quali si rimanda agli appositi capitoli.

 

  • Gestione fatture

  • Stampa fatture

  • Analisi fatture

  • Lista fatture

  • Contabilizzazione fatture

  • Analisi flusso documenti

  • Registrazione costi

  • Stampa etichette

 

La procedura consente di gestire più serie di Fatture, ciascuna delle quali richiede una specifica causale che governa una propria numerazione progressiva.
