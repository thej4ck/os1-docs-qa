Categorie

La procedura consente di definire delle categorie a cui associare gli articoli, che saranno utilizzate sul dispositivo per effettuare le ricerche degli articoli per categoria (ad albero).

Per ogni categoria deve essere specificato un CODICE ed una DESCRIZIONE, inoltre se la categoria fa parte di un raggruppamento deve essere indicata nella categoria di riferimento la categoria di appartenenza altrimenti la categoria di riferimento deve essere lasciata vuota. E' anche possibile indicare l'ordine di visualizzazione valorizzando il campo ORDINAMENTO.
