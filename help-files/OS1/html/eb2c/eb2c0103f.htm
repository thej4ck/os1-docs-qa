Utenti B2C

La tabella elenca gli utenti che si sono collegati all'area eCommerce B2C del web e si sono regolarmente registrati tramite la propria email; non è consentito inserire e/o cancellare i dati presenti, è possibile solo inserire o modificare il valore del campo "Codice cliente" che viene, solitamente, assegnato in automatico dalla fase di Importazione ordini web B2C.

Il codice cliente indica il codice con cui l'utente è stato inserito nell'anagrafica clienti aziendale.
