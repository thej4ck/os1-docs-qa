Dati consegna

La sezione video dati di consegna è attiva solo nel caso in cui la fattura sia di tipo immediato, informazione che viene indicata sulla causale, altrimenti non compare.

La sezione video dati di consegna consente l’inserimento delle istruzioni di consegna ed è situata in questa posizione perché la compilazione automatica di alcuni campi della successiva linguetta dati di fatturazione può essere influenzata dal contenuto dell’eventuale Destinazione diversa qui indicata.

Si ricorda infatti che la tabella destinazioni diverse dei clienti non contiene soltanto l’indirizzo di destinazione diversa, ma può contenere anche modalità di pagamento, domiciliazione bancarie, riferimento a listini, codici agenti diversi rispetto a quelli presenti nell’anagrafica del cliente intestatario della fattura.

In questo caso, i corrispondenti campi della finestra video dati di fatturazione vengono compilati con le informazioni presenti in anagrafica destinazione diversa anzichè con quelli presenti in anagrafica del cliente intestatario.

 

TIPO DESTINAZIONE: combo box per attivare l’eventuale inserimento di una destinazione diversa ed, in caso positivo, per indicare se la destinazione diversa è presente in anagrafica destinazioni o meno. Valori ammessi:

    • Nessuna: confermando, vengono ignorate tutte le informazioni relative alla destinazione.

    • Libera: confermando, si passa alla descrizione di una destinazione diversa occasionale.

    • Codificata: confermando, si passa al campo codice per scegliere una destinazione memorizzata per il cliente. Sul campo sono attive le funzioni di ricerca (F9) ed accesso rapido (CTRL+W).

CODICE: codice numerico della destinazione diversa, associata al cliente, ed a cui deve essere consegnato il prodotto. Il campo è visualizzato e significativo solo se il campo precedente ha il valore "Codificata". Sul campo sono attive le funzioni di ricerca (F9) ed accesso rapido (CTRL+W) per eventuali nuovi inserimenti o variazioni. A conferma, viene visualizzata la ragione sociale e l’indirizzo di destinazione diversa.

DESTINAZIONE DIVERSA 1: campo alfanumerico di 50 caratteri, abilitato solo nel caso in cui il campo tipo destinazione contiene il valore Libera, per descrivere la destinazione diversa.

DESTINAZIONE DIVERSA 2: campo alfanumerico di 50 caratteri, abilitato solo nel caso in cui il campo tipo destinazione contiene il valore Libera, per descrivere la destinazione diversa.

INDIRIZZO: campo alfanumerico di 50 caratteri abilitato, solo nel caso in cui il campo tipo destinazione contiene il valore Libera, per descrivere l’indirizzo della destinazione diversa.

CAP, LOCALITÀ, PROVINCIA: campi alfanumerici di 3, 50 e 2 caratteri, abilitati solo nel caso in cui il campo tipo destinazione contiene il valore Libera, per inserire i corrispondenti dati della destinazione diversa.                        

DESTINAZIONE AMMINISTRATIVA: campo visibile solo se previsto in configurazione standard; codice numerico della destinazione diversa di tipo sede amministrativa, associata al cliente. Sul campo sono attive le funzioni di ricerca (F9) ed accesso rapido (CTRL+W) per eventuali nuovi inserimenti o variazioni.

TIPO TRASPORTO: codice numerico presente nella tabella tipi di trasporto per definire le modalità di trasporto. Viene proposto, se presente, il codice esistente in anagrafica del cliente. Sul campo sono attive le funzioni di ricerca (F9) ed accesso rapido alla tabella (CTRL+W).

SPEDIZIONIERE 1: eventuale codice del primo spedizioniere, in caso di trasporto a mezzo vettore. Viene proposto, se presente, il codice esistente in anagrafica del cliente. Sul campo sono attive le funzioni di ricerca (F9) ed accesso rapido alla tabella (CTRL+W).

PROPRIETARIO AUTOMEZZO: nominativo del proprietario dell'automezzo che effettua il trasporto, sul campo sono attive le funzioni di ricerca (F9) ed accesso diretto alla tabella degli automezzi (CTRL+W). Il campo è presente solo se attiva la gestione della normativa antimafia.

TARGA: numero di targa dell'automezzo che effettua il trasporto, sul campo sono attive le funzioni di ricerca (F9) sulla tabella degli automezzi (CTRL+W). Il campo è presente solo se attiva la gestione della normativa antimafia.

SPEDIZIONIERE 2: eventuale codice del secondo spedizioniere, in caso di trasporto a mezzo vettore. Viene proposto, se presente, il codice esistente in anagrafica del cliente. Sul campo sono attive le funzioni di ricerca (F9) ed accesso rapido alla tabella (CTRL+W).

CODICE PORTO: eventuale codice delle modalità di  porto. Viene proposto, se presente, il codice esistente in anagrafica del cliente. Sul campo sono attive le funzioni di ricerca (F9) ed accesso rapido alla tabella (CTRL+W).

ASPETTO DEI BENI: campo alfanumerico per l’eventuale descrizione sintetica dell’aspetto dei beni. Sul campo sono attive le funzioni di ricerca (F9) ed accesso rapido alla tabella (CTRL+W).

ANNOTAZIONI: campo alfanumerico per eventuali annotazioni relative al trasporto.

MARCHE: campi alfanumerici di 30 caratteri che contengono le marche di spedizione del cliente; vengono proposte quelle inserite in tabella Marche per il cliente e/o la destinazione del documento. Sul campo 'Marche 1' sono attive le funzioni di ricerca (F9) ed accesso rapido alla tabella (CTRL+W).

BOTTONE ESTESE: attiva un campo note estese per indicare le marche estese del cliente; vengono  proposte quelle inserite in tabella Marche per il cliente e/o la destinazione del documento.
