Stampa certificazioni

Il programma permette la stampa della certificazione dei compensi erogati che deve essere inviata al percipiente.

La stampa può essere provvisoria oppure definitiva; in quest’ultimo caso a ciascun pagamento viene attribuito il numero di certificazione stampato sul documento; tale numero verrà utilizzato nel caso di ristampa di una certificazione già stampata.

 

DA PERCIPIENTE: indicare il codice del percipiente, con cui iniziare la stampa. Sul campo sono attive le funzioni di ricerca (F9).

A PERCIPIENTE: indicare il codice del percipiente, con cui terminare la stampa. Sul campo sono attive le funzioni di ricerca (F9).

ANNO: indicare l'anno dei movimenti. Il campo è obbligatorio.

DA DATA: indicare la data del movimento con cui iniziare la stampa. Il campo è obbligatorio.

A DATA: indicare la data del movimento con cui terminare la stampa. Il campo è obbligatorio.

DATA STAMPA: indica la data in cui viene effettuata la stampa. Viene proposta la data di sistema.

ORDINAMENTO: determina l'ordinamento in stampa: per data pagamento, per data movimento, per numero movimento.

INCLUDI CERTIFICAZIONI DA VERSARE: definisce se includere nella stampa anche le certificazioni non ancora versate.

STAMPA DEFINITIVA: definisce se la stampa è definitiva (casella con segno di spunta), in questo caso a ciascun pagamento viene attribuito il numero di certificazione stampato sul documento.

RISTAMPA CERTIFICAZIONE: se contiene la spunta consente di ristampare certificazioni già stampate in definitivo.

TIPO PROSPETTO: definisce quale prospetto aggiungere alla stampa; ritenute Irpef, ritenute Inps/Enasarco oppure entrambi.
