Stampa versamenti

La stampa dei versamenti riporta i dati del versamento effettuato. Nei parametri, oltre normali limiti di selezione quali numero ODL, anche singolo, data versamento o ODL, prodotto ecc. viene richiesta l'unità produttiva con proposta l'unità produttiva di default valida per l'utente connesso ad OS1; è possibile solo scegliere l'unità produttiva, se gestite in configurazione le unità produttive multiple.

 

Il pulsante Elenco consente una preselezione di stampa, mostrando nella finestra quali versamenti saranno stampati e consentendo, tramite la spunta delle caselle, di escluderli dalla stampa.
