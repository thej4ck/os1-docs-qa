Gruppi progetti

I gruppi progetti, codificati in base ad un codice alfanumerico di 3 caratteri, possono essere utilizzati per costruire raggruppamenti degli specifici progetti.

 

CODICE: codice alfanumerico di 3 caratteri per identificare il gruppo in esame. Sul campo sono attive le funzioni di ricerca (F9) sulla tabella stessa.

DESCRIZIONE: campo alfanumerico di 50 caratteri per descrivere il gruppo.

OBSOLETO: flag attivabile dall’utente per inibire il possibile utilizzo dell’elemento di tabella in esame nelle successive transazioni.
