Dati generali

 

CODICE: codice alfanumerico di 8 caratteri per identificare la categoria in esame. Sul campo sono attive le funzioni di ricerca (F9) sulla tabella stessa.

DESCRIZIONE: campo alfanumerico di 50 caratteri per descrivere la categoria.

BOTTONE LINGUE: attivo solo se in configurazione è prevista la gestione delle lingue,  accede interattivamente alla tabella Descrizioni in lingua delle categorie per consentire l’inserimento delle descrizioni in lingua estera.

CATEGORIA DI RIFERIMENTO: codice alfanumerico di 8 caratteri per identificare la categoria eventualmente associata a cui fare riferimento. Se la categoria di riferimento è vuota non sarà possibile associare, alla categoria attiva, nessun prodotto. Sul campo sono attive le funzioni di ricerca (F9) sulla tabella stessa.

NOTE: campo note, che sarà visibile nell'area eCommerce del web, che consente di descrivere in modo dettagliato la categoria.

BOTTONE LINGUE: attivo solo se in configurazione è prevista la gestione delle lingue,  accede interattivamente alla tabella Descrizioni estese in lingua delle categorie per consentire l’inserimento delle note in lingua estera.

N. ORDINAMENTO: numero d'ordine utilizzato per determinare l'ordinamento dei dati nella pagina web relativa all'elenco categorie.

OBSOLETO: flag attivabile dall’utente per inibire il possibile utilizzo dell’elemento di tabella in esame nelle successive transazioni.

Se presenti più modelli di eCommerce saranno presenti le caselle di spunta per indicare a quale tipo di eCommerce è associata la categoria.
