Generazione immagini per WEB

La procedura consente di effettuare una generazione automatica delle immagini per il WEB sia per le categorie, che per i prodotti. La generazione delle immagini avviene in base alle dimensioni e nella cartella presenti nella configurazione del modulo eCommerce.

Non sarà possibile effettuare la generazione se i parametri di configurazione non sono corretti oppure se è impostata una gestione manuale delle immagini.

Se presenti più modelli di eCommerce è presente la casella per indicare per quale tipo di eCommerce effettuare la generazione.

Dopo aver apposto la spunta alle caselle relative alle immagini da generare, categorie e/o prodotti,  la pressione del pulsante Esegui avvia la procedura di generazione.
