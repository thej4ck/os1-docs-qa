Accordi interbancari

La tabella consente di associare le banche per accordo interbancario; viene utilizzata dalla procedura di elaborazione distinta effetti.

 

CODICE: codice alfanumerico di 3 caratteri per identificare l’accordo in esame. Sul campo sono attive le funzioni di ricerca (F9) sulla tabella stessa.

DESCRIZIONE: campo alfanumerico di 50 caratteri per descrivere l’accordo interbancario.

BANCA: codice numerico relativo alla banca da includere nell'accordo interbancario. Sul campo sono attive le funzioni di ricerca (F9) sulla tabella stessa.
