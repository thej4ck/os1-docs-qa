Configurazione ordini fornitori

La compilazione di questa tabella è richiesta agli utenti che utilizzano il modulo Gestione Ordini Fornitori. Ciascun check box parametrizza l'abilitazione della richiesta della relativa data nei programmi di emissione ordini.

IMPORTA DESCRIZIONE ESTESA: Se attivo in fase di emissione ordine per i prodotti la cui descrizione estesa non è fissa viene riportata la descrizione estesa del prodotto nel primo campo di annotazioni disponibile sul rigo in modo tale da poterlo modificare.

TIPO EVASIONE RDA: Utilizzato nel programma di evasione RDA indica il criterio di evasione preferenziale che si intende utilizzare, attivando la pagina opportuna; valori ammessi: per n. richiesta, per prodotto, per data di consegna.

TIPO EVASIONE ORDINE: Utilizzato nel programma di evasione ordini, per evadere gli ordini aperti, indica il criterio di evasione preferenziale che si intende utilizzare, attivando la pagina opportuna; valori ammessi: per n. ordine, per prodotto, per data di consegna, per n. ordine (ordinata data/num decrescente).

STAMPA ORDINE PER MAGAZZINO DI CONSEGNA: Se spuntato, in fase di stampa ordine fornitore le righe vengono raggruppate e stampate per magazzino di consegna di rigo.

RIF. ORDINE SU DOCUMENTI: Definisce se si desidera inserire un rigo descrittivo, in fase di evasione ordine, che indica il documento evaso, valori ammessi:

    • Nessun riferimento: non genera alcun rigo descrittivo.

    • Riferimento interno: genera rigo descrittivo contenente l’indicazione della causale, del numero interno dell’ordine e della data dell’ordine.

    • Riferimento esterno: genera rigo descrittivo contenente il riferimento inserito nel campo “Vs. richiesta” sulla testa dell’ordine.

    • Entrambi i riferimenti: genera rigo descrittivo con entrambi i riferimenti.

DESCRIZIONE PER DOCUMENTO: Breve descrizione, utilizzata in fase di evasione ordine fornitore nel caso in cui si desideri inserire il riferimento; precede il riferimento all’ordine stesso.

CONTROLLO PREZZO IN DDT: Se attivo in fase di registrazione Ddt, in assenza del modulo Gestione acquisti, viene segnalato se il rigo ha prezzo uguale a zero.

Controllo prezzo in ordine: Se attivo in fase di inserimento ordine viene segnalato se il rigo ha prezzo uguale a zero.

STAMPA ORDINE RAGGRUPPATO PER PRODOTTO: Se spuntato, in fase di stampa ordine fornitore le righe vengono raggruppate e stampate per prodotto.
