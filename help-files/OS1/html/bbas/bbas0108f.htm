Codici Iso

La tabella consente di memorizzare i Codici ISO delle nazioni aderenti alla Comunità Europea. La tabella viene precaricata in sede di installazione ed è completa di tutti le nazioni aderenti alla Comunità al momento dell’installazione. L’accesso a questo programma è quindi circoscritto al caso di adesione di nuove nazioni alla CEE.

Finestra di input in formato dettaglio

Finestra di input in formato tabella

 

 

CODICE: codice alfanumerico di 2 caratteri, per individuare la nazione. Sul campo sono attive le funzioni di ricerca (F9) sulla tabella stessa.

DESCRIZIONE: campo alfanumerico di 50 caratteri per contenere la descrizione della nazione in esame.
