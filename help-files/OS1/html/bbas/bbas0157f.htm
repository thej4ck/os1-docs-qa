Schemi contabilizzazione

La tabella viene fornita precaricata e consente di definire le possibili registrazioni contabili applicabili in fase di contabilizzazione distinta pagamenti fornitori. Il codice schema potrà essere referenziato in anagrafica conto corrente (vale per tutti i tipi di distinta), oppure a livello di condizioni. Per il funzionamento delle varie casistiche si rinvia agli esempi forniti all’interno della tabella stessa, dove nel campo NOTE è indicato per ogni schema l'esempio di come sarà rappresentata la scrittura contabile.
