Voci di costo

Si tratta di una tabella che consente di creare e memorizzare le Voci di costo, definendone anche l’eventuale divisa di gestione. Le voci di costo possono essere operative nelle procedure di Gestione della Produzione e di Gestione magazzino. La sua creazione può pertanto essere omessa dagli utenti che non hanno esigenze in merito.

 

CODICE: codice alfanumerico di 3 caratteri per identificare la Voce di costo in esame. Sul campo sono attive le funzioni di ricerca (F9) sulla tabella stessa.

DESCRIZIONE: campo alfanumerico di 50 caratteri per descrivere la voce  di costo.

GRUPPO VOCI: codice alfanumerico di 3 caratteri presenti nella Tabella Gruppi voci di costo per definire a quale eventuale gruppo appartiene la voce di costo in esame. Sul campo sono attive le funzioni di ricerca (F9) sulla tabella stessa.

GESTIONE DIVISA: combo box per definire le modalità di addebito della voce di costo attiva. Valori ammessi: Divisa di conto (Euro o Lire); Divisa estera.

RICHIESTA FORNITORE: definisce se in fase di inserimento dei dati deve essere richiesto il codice del fornitore (segno di spunta), oppure no (casella vuota).

OBSOLETO: flag attivabile dall’utente per inibire il possibile utilizzo dell’elemento di tabella in esame nelle successive transazioni.
