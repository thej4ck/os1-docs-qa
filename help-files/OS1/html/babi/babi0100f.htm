Modelli di riclassificazione

 

CODICE: codice alfanumerico di 3 caratteri per identificare il modello in esame. Sul campo sono attive le funzioni di ricerca (F9) sulla tabella stessa.

DESCRIZIONE: campo alfanumerico di 50 caratteri per descrivere il modello.

NUMERO DECIMALI: definisce se gli importi dei valori relativi alle voci appartenenti a questo modello devono essere arrotondati all'unità di Euro oppure devono essere gestiti con i decimali previsti per la divisa Euro.

OBSOLETO: flag attivabile dall’utente per inibire il possibile utilizzo dell’elemento di tabella in esame nelle successive transazioni.
